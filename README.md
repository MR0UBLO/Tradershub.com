# Tradershub.com Backend

**A comprehensive trading simulation platform with real-time market engines, AI signals, and M-Pesa integration.**

## Features

- 🔐 **Authentication** - JWT-based user registration and login
- 💰 **Wallet System** - Balance tracking and transaction history
- 📊 **Trading Engine** - Open/close trades with profit/loss calculation
- 🎯 **Market Simulation** - Real-time price updates and candle generation
- 🔔 **WebSocket Support** - Live market data and trade updates
- ⚙️ **Background Jobs** - Automated market updates and trade settlement
- 📱 **M-Pesa Integration** - Mobile money deposits and withdrawals
- 🛡️ **Security** - CORS, helmet, rate limiting, input validation

## Quick Start

```bash
npm install
cp .env.example .env
npm run prisma:migrate
npm run dev
```
